# Security Policy

## Supported Versions

Use this section to tell people about which versions of your project are
currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.8.x   | :white_check_mark: |

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability within this project, please help us maintain the security of our users' data by reporting it privately. 

To report a vulnerability:
1. Email: mathieugfortin@gmail.com
2. Provide a brief description of the issue and steps to reproduce it.

We will acknowledge your report within 7 days and provide a timeline for a fix. Once the fix is deployed, we will give you credit for the discovery (unless you prefer to remain anonymous).
